# Example: Alternative Implementation

## Scenario

You want to see how a different AI model would implement the same logic.

## Prompt

```
/ask-oracle What's a more efficient way to implement the caching logic in src/cache.py?
```

## What Happens

1. Oracle reads the current implementation
2. Asks Codex for alternative approaches
3. Compares the suggestions with the current code
4. Reports alternatives with trade-off analysis
